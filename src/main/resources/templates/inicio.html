Sistema de Gestión de Movimientos

El Sistema de Gestión de Movimientos busca ser un software encargado de registrar los movimientos de una empresa con el fin de permitir un registro de inventarios para que los usuarios puedan tener una ayuda contable con respecto a sus entradas y salidas.

Este sistema se desarrolla como proyecto final del curso de Desarrollo de Software de la Universidad de Antioquia en Convenio con el Icetex, por el grupo 4.
